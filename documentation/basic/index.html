    Basics¶

    There are 32 services for this applications. The different types of service calls, their parameters and responses are described below.

    Acknowledgement¶

    Acknowledgement data will be present in all the responses. The status will be the overall status of the request action. The ‘message’ will be the message from the server that can be displayed to the end users. The structure will be:

    export interface AcknowledgementResponse {
    acknowledgement: Acknowledgement
}

export interface Acknowledgement {
    message: string
    status:  string
}

    {
    "acknowledgement": {
        "message":"Categories successfully populated",
        "status":"SUCCESS"
    }
}
