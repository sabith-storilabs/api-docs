    Contents of a combo product¶

    This service is to get the contents of a particular combo product. This service has the same response of the product listing service.

    Request details

    API Endpoint : /products/{productId}/related/contents
    Request type : GET
    Payload : None

    Object Details¶

    • productListInfo - This object will have the meta information about the list being populated. Values include the total rating out of which the individual rating will be based on, minimum price the cart total should be to proceed to checkout and the maximum quantity of a single SKU that can be added to the online/offline cart.
    • product: This array will hold the list of products.
      • brand: This object will hold the details of the brand of the product.
      • rating: This object will hold a summary of the product ratings.
      • skus: This object will hold the information of the product SKUs.
